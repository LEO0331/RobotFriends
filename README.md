# RobotFriends

RobotFriends is a simple, fast website for discovering and searching robot profiles.

## Why People Use It

- Browse a clean list of robot cards
- Search instantly by robot name
- Enjoy a responsive, lightweight experience on desktop and mobile

## Live Site

- [RobotFriends on GitHub Pages](https://leo0331.github.io/RobotFriends/)

## What You Can Do

- Open the site and view robot profiles
- Type in the search box to filter results in real time
- Scroll through results smoothly

## Tech Overview

- React frontend
- Data fetched from JSONPlaceholder
- Hosted on GitHub Pages with GitHub Actions deployment

## Local Setup

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Build for Production

```bash
npm run build
```

## Project Goal

Deliver a friendly demo app that showcases fast search, clean UI, and simple deployment.
